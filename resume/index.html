Previous Jobs

These are some of the places where I’ve worked at and what I’ve done there.

Sonata Services
Worked from September 2013 to January 2014.

Did mostly Systems Administration jobs. My main role was to create and configure EC2 instances with preconfigured AMI images through Chef, basically I was automating the creating and configuration of entire environments for the applications we served, this included staging and production servers as well as Database servers, all hosted on AWS and pulling the code bases from Github.

Softtek
Worked from February 2013 to November 2013.
I worked on the Application Management team where I was mainly monitoring and giving back-end support to several in-house applications, this included managing different servers with different OSes, e.g. Solaris, RedHat, Windows Server 2008, Windows Server 2003.
Also I managed and supported Oracle databases, this included generating custom reports, analyzing the routines and submitting improvement suggestions to said routines (stored procedures, packages, materialized views, etc.) as well as designing test cases for the different improvements applied to the databases. The main tools used in my job were PL/SQL, Bash Scripting, Python (for a process improvement I worked on to submit), Remote Desktop, SSH (via PuTTY) and a set of proprietary tools designed specifically for the back-end management.

ArteDigital
Worked from January 2011 to July 2012.
My main role here was that of a Software Quality Agent, performing different tests to two different virtual communities, one created on Drupal and the other in a custom in-house CRM application. Besides these activities, I was responsible for the management and monitoring of the network, server maintenance and workstation maintenance as well. I was also in charge of the configuration of new services in the network, data base management and code repository management (GIT & SVN).
